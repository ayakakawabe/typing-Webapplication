# 概要
実験に使うために作成したタイピング問題のWebアプリケーション

タイピング問題は3種類で、正答率と回答速度を測ることができる

タイピングで使用できるキーは半角英数字・ー（長音記号）

# 使用方法
事前にNode.jsのインストールしておく
1. githubからコードをローカル環境にダウンロード、またはクローンする
2. typing_webAPPフォルダに移動
   ```sh
   cd ./typing_webAPP
   ```
4. 依存パッケージをインストール
   ```sh
   npm install
   ```
6. ローカルサーバを立てる
   ```sh
   npm run dev
   ```
[URL](http://127.0.0.1:5173/)にアクセス
